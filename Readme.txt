Requerimientos:
-Se debe instalar Python 3.6
-Se debe instalar Pygame 1.9.3

Instalación de Pygame:

1-Abra Mi PC
2-Ingrese al disco duro
3-En la esquina superior derecha escriba "Python"
4-Al cargar los resultados, ingrese en la carpeta "Python 36-32"
5-Ingrese en la carpeta "Scripts"
6-Coloque en esta carpeta el archivo de Pygame incluido en la carpeta del juego
7-Presione Shift + Click Derecho en algun espacio en blanco de la carpeta
8-Presione la opción de "Abrir ventana de comandos aquí"
9-Escriba en la barra de comandos "pip3 install pygame-1.9.3-cp36-cp36m-win32.whl"

En caso de que se complique la instalación, existen tutoriales más explícitos en Youtube.

Ejecución del programa:

Para ejecutar el programa es suficiente con dar doble click sobre el archivo "Main.py" que está en la carpeta "ProyectoFinal". 
Todos los archivos incluidos deben estar dentro de una misma carpeta para tener un funcionamiento óptimo.
